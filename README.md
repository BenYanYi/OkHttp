# OkHttp
## OkHttp请求封装
### 使用方法

module 下添加

     compile 'com.github.love-yanyi:OkHttp:1.1.3'

<br/>
project 下添加

    allprojects {
        repositories {
            jcenter()
            maven {
                url 'https://jitpack.io'
            }
        }
    }

<br/>

如果需要使用Jlog，可以查看https://github.com/love-yanyi/Logger
